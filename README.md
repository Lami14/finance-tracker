# Finance Tracker

A personal finance tracker web app built with HTML, CSS, and JavaScript.  
Track your income and expenses, categorize transactions, and visualize spending with a pie chart.



 Features

 Add income and expenses with description and amount
Categorize transactions (Food, Transport, Bills, Shopping, Salary, Other)
Delete transactions
 Calculate balance, total income, and total expenses
 Data saved in localStorage (persists after refresh)
 Visualize spending using Chart.js pie chart
Responsive and user-friendly UI
